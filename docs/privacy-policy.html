OneClick Wellness / 壹鍵康

私隱政策

1. 簡介

歡迎使用由 OneClick HealthTech Limited（下稱「本公司」或「我們」）透過 OneClick Wellness／壹鍵康應用程式及相關網站提供的服務（下稱「本服務」）。我們重視您的私隱，並依據香港《個人資料（私隱）條例》（下稱「PDPO」）處理個人資料。使用本服務即表示您已閱讀並理解本政策；若不同意，請勿使用本服務。

本政策適用於目前主要服務香港用戶；若您預約香港或大灣區／內地之合作體檢中心，下文第 6、7 節說明向體檢機構披露及跨境／境外處理。

2. 資料控制者及聯絡方式

  • 資料控制者：OneClick HealthTech Limited
  • 私隱查詢／行使權利：info@oneclick.hk（主旨請註明「私隱查詢」或「Personal Data Request」）
  • 資料刪除：請參閱 用戶資料刪除說明

3. 我們收集的資料類型

按您使用情況，我們可能處理以下類別（未必全部適用於每位用戶）：

  • 帳戶及聯絡資料：姓名、電郵、電話、性別、帳戶識別符、第三方登入識別（如 Firebase／Facebook，視您選擇的登入方式而定）。
  • 服務及交易資料：體檢與保健品訂單、付款狀態（卡資料由 Stripe 等 PCI 合規機構處理，我們通常不儲存完整卡號）、預約時段、分店、禮券選擇與寄送地址／電郵、優惠與行銷同意紀錄。
  • 健康與問卷內容：您自願填寫的健康問卷、生活習慣與相關輸入；以及您主動上傳的體檢報告或醫學相關檔案（上傳前 App 會另行提示同意）。
  • 裝置與使用情況：裝置型號、作業系統、App 版本、日誌、安全性相關資料、Cookie 或同類技術（Web 版）、IP 位址。
  • 通訊內容：您與客服、AI 助理的訊息（僅於提供該服務所需範圍內）。

4. 我們如何使用個人資料

我們將資料用於：

  • 提供、維持及改進本服務（註冊、登入、體檢／保健品目錄、預約與結帳）；
  • 履行預約及訂單，包括向合作體檢中心轉交預約所需資料（見第 6 節）；
  • 根據問卷、健康檔案及（如適用）體檢報告摘要，提供健康管理參考及產品推薦（非醫療診斷，您仍應諮詢合資格醫護人員）；
  • 客戶支援、保安、欺詐防範及遵守法律或監管要求；
  • 在取得同意或 PDPO 允許下，發送服務通知或行銷（您可隨時退出行銷）。

5. 保留期間

  • 帳戶及問卷資料：直至您要求刪除帳戶或停止使用，並完成我們的刪除流程；
  • 訂單、預約、付款及醫療相關交易紀錄：最長 7 年（會計、稅務及潛在爭議需要），屆期刪除或匿名化；
  • 技術日誌及保安紀錄：一般較短（例如 90 日至 12 個月，視用途而定）；
  • 法例要求更長保留者，以法例為準。

6. 向合作體檢中心及服務供應商披露

6.1 合作體檢中心（香港及大灣區／內地）

當您預約體檢，我們會向該次服務的合作醫療／體檢機構提供履約所需資料，可能包括：英文／中文姓名、性別、電話、電郵、預約日期時間、分店、所選套餐名稱、禮券寄送資料（如適用）及您於預約表填寫的聯絡資料。

若您預約大灣區／內地體檢中心（例如深圳灣粵美特等），上述資料將提供予位於內地的合作機構，以便安排體檢。我們不會自動向體檢中心傳送您完整健康問卷內容，除非營運流程另有需要且已另行告知。

您提交預約並完成付款，即表示理解上述披露為提供服務所必需。

6.2 服務供應商（代處理者）

我們委任以下類別供應商代為處理資料（僅限履行職能所需）：

  • MongoDB Atlas — 資料庫託管；
  • Amazon Web Services (S3) — 檔案及圖片儲存；
  • Vercel — 應用程式及 API 託管；
  • Firebase (Google) — 用戶身份驗證；
  • Stripe — 網上付款；
  • Google Cloud — AI／文字嵌入及相關雲端服務（用於內容處理及推薦，如已啟用）；
  • 電郵／通訊 — 客戶通知及支援。

我們會要求供應商採取合理保安措施；詳細清單由本公司內部維護並定期檢視。

7. 跨境及境外資料傳送

您的個人資料可能儲存於香港以外的雲端伺服器（例如 MongoDB Atlas、AWS 所在地區）。此外，如第 6.1 節所述，預約內地體檢時，資料會提供予內地合作機構。我們會採取合約及技術上合理措施，並依 PDPO 及適用指引處理跨境／境外傳送。

8. Cookie 與類似技術

網頁版或嵌入元件可能使用 Cookie 或同類技術，以維持登入、喜好及流量分析。您可透過瀏覽器設定管理（可能影響部分功能）。

9. 保安

我們採取合理技術及組織措施，包括 HTTPS 傳輸、存取控管及權限分割。任何系統均無法保證絕對安全，請妥善保管您的帳戶及裝置。

10. 您在 PDPO 下的權利

您有權：

  • 查閱我們持有的您的個人資料；
  • 改正不準確的資料；
  • 刪除帳戶及相關資料（見 資料刪除說明，部分紀錄可能因法律須保留）；
  • 就處理提出查詢或投訴。

請電郵至 info@oneclick.hk；我們可能要求核實身份，並於 PDPO 規定的合理期限內回覆（一般不多於 40 日）。您亦可向香港個人資料私隱專員公署投訴。

11. 兒童

本服務不擬供未滿 18 歲人士使用。如發現兒童在未獲適當同意下提供資料，我們將依法處理。

12. 本政策變更

我們可更新本政策；重大變更將於 App 或網站公布。請定期查閱本頁底部之生效日期。

13. 聯絡我們
