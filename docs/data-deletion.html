OneClick Wellness / 壹鍵康

用戶資料刪除說明

1. 概要

若您曾使用 OneClick Wellness（壹鍵康）應用程式或相關服務，並希望刪除或停止使用與您帳戶相關的個人資料，可依照以下方式提出請求。我們會在核實身份後，於適用法律及合理範圍內處理。


2. 提出刪除／停用請求

請使用您於本服務註冊或常用之電郵地址，寄信至：

info@oneclick.hk

郵件建議包含：

  • 主旨註明「資料刪除」或「Data deletion request」
  • 您的帳戶識別（例如註冊電郵或手機號碼，與 App 內一致）
  • 簡述希望刪除的範圍（例如刪除整個帳戶及相關紀錄）


3. 身份核實

為保障帳戶安全，我們可能要求您提供額外資料以核實身份；若無法核實，可能無法完成刪除。


4. 處理時間

在核實身份後，我們會在合理期間內處理；部分資料可能因法律、會計、爭議解決或保安需要而須保留較長時間，屆時會於法律允許範圍內刪除或匿名化。


5. 第三方登入（例如 Facebook）

若您曾使用 Facebook 等第三方登入，除依本頁向本服務提出請求外，亦可於該第三方平台檢視或撤銷應用程式之授權（以該平台設定為準）。


6. 詳細私隱條款

有關個人資料的收集與使用，請參閱 私隱政策。
